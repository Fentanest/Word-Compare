# Word-Compare: Word 문서 비교 도구

## 소개

이 도구는 두 개의 Microsoft Word 문서를 비교하여 변경 사항을 시각적으로 보여주는 데스크톱 애플리케이션입니다. 원본 문서와 수정된 문서를 나란히 놓고 비교하여 차이점을 강조한 새로운 Word 문서를 생성합니다. 또한, 변경된 내용을 문단별로 정리하고 수정된 단어를 시각적으로 강조한 상세한 Excel 보고서를 생성하는 강력한 기능을 제공합니다.

## 주요 기능 요약

-   **직관적인 UI**: 드래그 앤 드롭으로 누구나 쉽게 비교할 파일을 추가하고 관리할 수 있습니다.
-   **정확한 비교**: PC에 설치된 Microsoft Word와 직접 연동하여, Word의 강력한 '문서 비교' 기능으로 정확한 결과를 보장합니다.
-   **두 가지 결과물**: 모든 변경 내용이 추적된 '비교 결과 Word 파일'과, 수정된 단어까지 상세히 표시하는 'Excel 보고서'를 모두 생성할 수 있습니다.
-   **상세 분석 보고서**: Excel 보고서는 삭제된 단어는 <span style="color:blue; text-decoration:line-through;">파란색+취소선</span>으로, 추가된 단어는 <span style="color:red; font-weight:bold;">빨간색+굵게</span>로 표시하여 변경 내용을 한눈에 파악하게 해줍니다.
-   **일괄 처리**: 여러 개의 문서 쌍을 목록에 추가하여 한 번에 비교 작업을 수행할 수 있습니다.

## 상세 기능 설명

이 프로그램은 크게 **사용자 인터페이스 및 파일 비교를 담당하는 부분**과 **상세 엑셀 보고서를 생성하는 부분**으로 나뉩니다.

### 1. 메인 프로그램 기능

사용자가 보는 화면과 프로그램의 핵심 동작을 제어합니다.

-   **GUI (그래픽 사용자 인터페이스)**: 사용자가 마우스로 쉽게 파일을 추가하고 버튼을 누를 수 있는 직관적인 화면을 제공합니다.
-   **파일 관리 기능**:
    -   **드래그 앤 드롭**: 탐색기에서 Word 파일을 마우스로 끌어 '전(Before)' 목록과 '후(After)' 목록에 쉽게 추가할 수 있습니다.
    -   **목록 편집**: 목록 안에서 파일 순서를 바꾸거나, `Delete` 키를 눌러 선택한 파일을 삭제할 수 있습니다.
-   **핵심 비교 기능 (MS Word 연동)**:
    -   **필수 조건**: 이 기능은 사용자의 PC에 **Microsoft Word가 반드시 설치되어 있어야 동작합니다.**
    -   보이지 않는 배경에서 MS Word 프로그램을 실행시켜, Word가 자체적으로 제공하는 강력한 `문서 비교` 기능을 실행하여 두 문서의 차이점을 찾아냅니다.
-   **결과물 생성**:
    -   **비교 결과 Word 파일**: Word의 비교 기능으로 만들어진 결과(모든 변경 사항이 표시된 문서)를 `비교_결과_`라는 이름이 붙은 새 Word 파일로 지정된 경로에 저장합니다.
    -   **엑셀 보고서 생성 (선택 사항)**: 사용자가 '엑셀 보고서 생성' 옵션을 체크하면, 상세 보고서 생성을 지시합니다.
-   **사용자 편의 기능**:
    -   **실시간 로그**: 파일 열기, 비교, 저장 등 모든 작업 진행 상황을 화면 하단의 로그 창에 표시하여 프로그램이 어떤 작업을 하고 있는지 알려줍니다.
    -   **저장 경로 설정**: 사용자가 결과물을 저장할 폴더를 직접 지정할 수 있으며, 프로그램은 해당 경로를 기억했다가 다음 실행 시 자동으로 불러옵니다.

### 2. 엑셀 보고서 생성 기능

두 문서의 텍스트 차이점을 분석하여 상세한 Excel 보고서를 만드는 역할을 전담합니다.

-   **텍스트 추출**: 원본과 수정본 Word 문서에서 눈에 보이는 모든 문단 텍스트를 추출합니다.
-   **다단계 분석**: 먼저 **문단 단위**로 비교하여 변경된 문단을 찾아낸 뒤, 해당 문단 내에서 다시 **단어 단위**로 정밀하게 비교하여 실제 수정된 단어를 식별합니다.
-   **시각적 보고서 생성**:
    -   보고서는 '위치', '수정 전', '수정 후' 세 개의 열로 구성됩니다.
    -   단순히 텍스트만 복사하는 것이 아니라, 셀 안에서 변경된 단어에 다음과 같이 **특수 서식을 적용**하여 차이점을 한눈에 파악할 수 있게 합니다.
        -   **삭제된 단어**: '수정 전' 열에 <span style="color:blue; text-decoration:line-through;">파란색, 취소선</span>으로 표시됩니다.
        -   **추가된 단어**: '수정 후' 열에 <span style="color:red; font-weight:bold;">빨간색, 굵은 글씨</span>로 강조됩니다.
-   **예외 처리**: 만약 두 문서의 텍스트 내용에 아무런 차이가 없다면, "변경 사항 없음" 로그를 남기고 불필요한 빈 Excel 파일은 만들지 않고 자동으로 삭제합니다.

## 요구 사항

-   **운영체제**: Windows
-   **필수 소프트웨어**: Microsoft Word

## 사용 방법

1.  `WordCompare.exe`와 같은 실행 파일을 실행합니다.
2.  화면 왼쪽의 **'전(Before)'** 목록에 원본 Word 파일을, 오른쪽 **'후(After)'** 목록에 수정된 Word 파일을 드래그 앤 드롭으로 추가합니다.
3.  두 목록의 파일 개수는 동일해야 하며, 같은 순서에 있는 파일끼리 비교가 진행됩니다.
4.  **'저장 경로'** 찾아보기 버튼을 눌러 비교 결과 파일이 저장될 폴더를 선택합니다. (기본 경로는 바탕화면입니다.)
5.  상세 분석 보고서를 받으려면 **'엑셀 보고서 생성'** 체크박스를 선택합니다.
6.  **'비교 시작'** 버튼을 클릭하여 작업을 시작합니다.
7.  작업이 완료되면 지정된 저장 경로에서 결과 파일(`비교_결과_...docx`, `변경내용_....xlsx`)을 확인할 수 있습니다.
